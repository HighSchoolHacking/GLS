# Print Statement

Creates a print statement.

## Commands

### `print`

`print : foo`

Outputs a print statement that prints the parameter.

## Usage

```
print : foo
```

### CSharp

```csharp
Console.WriteLine(foo);
```

### Java

```java
System.out.println(foo);
```

### Python

```python
print(foo)
```

### Ruby

```ruby
puts foo
```

### TypeScript

```typescript
console.log(foo)
```

### JavaScript

```javascript
console.log(foo)
```
