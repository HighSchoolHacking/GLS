# Return Statement

A statement used to return a value from a function.

## Commands

### `return`

`return : foo`

Outputs a statement that returns the passed parameter.

## Usage

```
return : foo
```

### CSharp

```csharp
return foo;
```

### Java

```java
return foo;
```

### Python

```python
return foo
```

### Ruby

```ruby
return foo
```

### TypeScript

```typescript
return foo;
```
