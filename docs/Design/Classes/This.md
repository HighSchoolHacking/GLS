# This Keyword

Keyword for an object to refer to itself.

## Commands

### `this`

`this`

Outputs the keyword an object in a language uses to refer to itself.

## Usage

```
this
```

### CSharp

```csharp
this
```

### Java

```java
this
```

### Python

```python
self
```

### Ruby

```ruby
self
```

### TypeScript

```typescript
this
```
